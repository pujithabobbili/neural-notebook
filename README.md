# The Neural Notebook

A modern data science & AI blog built with Next.js, Tailwind CSS, and Lucide icons.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the blog.

## Tech Stack

- **Next.js 16** (App Router)
- **Tailwind CSS v4**
- **Lucide React** icons
- **TypeScript**
